Tabuleiro4 é um jogo musical interativo que deve ser usado em conjunto um
Ukulele, o objetivo é avançar no tabuleiro tentando chegar ao final dos seus
colegas, realizando as tarefas selecionadas pelo jogo.

Requisitos:
O projeto foi feito no Xcode, e designado para a versão 12.0 do iOS, alguns
bugs e reparos ainda precisam ser corrigidos.

Para começar a editar o projeto, basta clonar o repositório e abrir o arquivo
Tabuleiro4.xcodeproj no Xcode.
